1.  **Install packages**

    Run command:

    ```shell
    yarn install
    ```

1.  **Create components**

    This command creates components in the `./src/components` folder

    ```shell
    yarn generate
    ```

1.  **Test components**

    This command will look for errors within the `./src` folder

    ```shell
    yarn lint
    ```
